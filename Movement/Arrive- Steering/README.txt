This folder contains the java and class files for Arrive behaviour as well as the screenshots of the program.

There are three classes used for the implemetation of this program: Steering_Variables, CharS and Arrive_Steering.

Steering_Variables contains the data structure holding all the steering variables.
CharS contains all the functions of display(), update(), breadCrumbs() and 2 of the arrive functions.
Arrive_Steering is the main class calling all the functions. 

One of the arrive functions is commented out in Arrive_Steering. Uncomment the function whose working you wish to view.
