This folder contains the java and class files for Wander behaviour as well as the screenshots of the program.

There are three classes used for the implemetation of this program: Steering_Variables, CharWander and Wander_Steering.

Steering_Variables contains the data structure holding all the steering variables.
CharWander contains all the functions of display(), update(), breadCrumbs() and 2 of the wander functions.
Wander_Steering is the main class calling all the functions. 

One of the wander functions is commented out in Wander_Steering. Uncomment the function whose working you wish to view.
