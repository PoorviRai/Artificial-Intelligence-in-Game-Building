This folder contains the java and class files for Kinematic Motion as well as the screenshots of the program.

There are three classes used for the implemetation of this program: Kinematic_Variables, CharK and Basic_Motion.

Kinematic_Variables contains the data structure holding all the kinematic variables.
CharK contains all the functions of display(), update() and breadCrumbs().
Basic_Motion is the main class calling all the functions.
