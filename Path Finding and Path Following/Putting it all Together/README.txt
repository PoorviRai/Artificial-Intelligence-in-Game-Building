This folder contains the java and class files for Path Following as well as the screenshots of the program.

There are three classes used for the implemetation of this program: Steering_Variables, IndoorEnv, Follow_Path and MainMovement.

Steering_Variables contains the data structure holding all the steering variables.
IndoorEnv contain the function of displayObstacles().
Follow_Path contains all the functions of display(), update(), arrive1() and breadCumbs().
MainMovement is the main class calling all the functions and contains settings(), setup(), draw() and mousePressed() functions.
